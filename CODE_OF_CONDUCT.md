# Code Of Conduct

KeiPix 需要一个能认真讨论技术、设计和风险的协作环境。请在 issue、PR、review 和讨论中保持清晰、克制、可复现。

## Expected Behavior

- 尊重不同经验背景的贡献者。
- 批评代码、设计和证据，不攻击个人。
- 讨论问题时尽量给出复现步骤、截图、日志或代码位置。
- 对安全、账号、R-18/R-18G、远端写入等敏感主题保持谨慎，不公开他人隐私或凭据。
- 接受维护者为了项目边界、License、安全或 QA 要求而延后或拒绝改动。

## Unacceptable Behavior

- 人身攻击、骚扰、歧视性语言或持续挑衅。
- 公开他人的 token、cookie、账户信息、私密截图或下载内容。
- 鼓励绕过授权、批量滥用 Pixiv 接口或规避安全边界。
- 在未说明来源和 License 的情况下搬运第三方源码。

## Enforcement

维护者可以编辑或删除不合适内容、锁定讨论、关闭 issue/PR，或限制持续破坏协作环境的账号参与。安全或隐私问题会优先处理。
